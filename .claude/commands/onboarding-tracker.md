# Onboarding Progress Tracker

Comprehensive tracking and validation system to ensure successful workspace onboarding with personalized guidance and achievement recognition.

## Progress Tracking Commands

```bash
# Check onboarding progress
/project:onboarding-status
/project:onboarding-progress

# Complete specific onboarding steps
/project:complete-step --step=<step-id>
/project:validate-step --step=<step-id>

# Reset or resume onboarding
/project:resume-onboarding
/project:reset-onboarding --confirm

# Get personalized next steps
/project:next-steps
/project:onboarding-help --stuck
/project:onboarding-celebrate

# Export onboarding achievements
/project:export-achievements
/project:share-progress --team
```
